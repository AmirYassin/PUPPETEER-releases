# PUPPETEER

**macOS AI Swarm Orchestration System**

Desktop-native multi-agent orchestrator with a Liquid Silver aesthetic. Compile your AI workforce into a single, beautiful app.

## Download

Go to [**Releases**](https://github.com/AmirYassin/PUPPETEER-releases/releases) and download the latest `.dmg`.

## Install

1. Download `PUPPETEER_v1.0.15.dmg`
2. Open the DMG
3. Drag `PUPPETEER.app` to Applications
4. Launch from Applications

## Requirements

- macOS 15.0 (Sequoia) or later
- Apple Silicon (arm64)

## Features

- AI-powered multi-agent swarm orchestration (Gemini LLM)
- Liquid Silver 3D logo (Three.js WebGL)
- Analog Paper aesthetic (Courier New, Moleskine Cream)
- Agent conversation UI, task management, memory viewer
- Inter-agent communication channels
- Permission-gated human-in-the-loop approval
- Native compilation, signed & notarized
